# usercenter
简单的用户中心前后端分离项目（主要用户熟悉前后端开发流程）

参考来源：https://docs.qq.com/doc/DUG93dVNHbVZjZXpo

## 技术选型
### 前端
主要运用阿里 Ant Design 生态：
- HTML + CSS + JavaScript 三件套
- React 开发框架
- Ant Design Pro 项目模板
- Umi 开发框架
- Umi Request 请求库

### 后端
- Java 编程语言
- Spring + SpringMVC + SpringBoot 框架
- MyBatis + MyBatis Plus 数据访问框架
- MySQL 数据库
- jUnit 单元测试库
